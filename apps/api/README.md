# apps/api

AIryss バックエンド API（FastAPI 予定）

## 概要

- フレームワーク: FastAPI
- 言語: Python 3.12+
- ORM: SQLAlchemy (async)
- マイグレーション: Alembic

## セットアップ（準備中）

```bash
# 仮想環境作成
python -m venv .venv
source .venv/bin/activate

# 依存インストール
pip install -r requirements.txt

# 開発サーバー起動
uvicorn main:app --reload
```

> ※ 現在はプレースホルダです。FastAPI プロジェクト初期化前の状態です。
